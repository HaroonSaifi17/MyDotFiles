## Installation

To get started, follow these steps:

1. Clone this repository to your local machine.

```bash
git clone https://github.com/haroonsaifi17/MyDotFiles && cd MyDotFiles
```

2. setup

```bash
chmod 777 ./setup.sh && ./setup.sh
```
